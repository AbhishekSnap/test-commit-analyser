# GIT Digest
## Hi, This is GitHub Commit Analysis Tool. 
### Author: Abhishek Kumar and Calvin Fuss
### Date: 12 April 2026
### Role: AI Consultant
### Organisation: Snap Analytics, United Kingdom

#### Description: A lightweight automation tool that uses AI to explain every code commit your team makes - automatically. Every time a developer pushes code, it reads what changed, sends it to Claude AI for analysis, and writes a plain English summary to your repo's Wiki. No manual reporting, no chasing updates. Your team's daily work documents itself. Built on the Model Context Protocol (MCP), it works as a plug-and-play GitHub Action - any team adds one file to their repo and it just works.

### What gets logged for every commit:

1. Who changed what and when
2. Plain English explanation of the change
3. Technical summary for developers
4. Type of change and commit quality score

#### Built with: Python · MCP · Claude AI · GitHub Actions · GitHub Wiki
